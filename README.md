# Navi - Bioluminescent E-commerce Concept

This project represents the visual concept and architecture for **Navi**, a holistic e-commerce ecosystem.

## viewing the Prototype
1.  Run `npm install` (if you haven't already).
2.  Run `npm run dev`.
3.  Open the provided localhost URL.

## Documentation
*   [Design Concept & Architecture](./DESIGN_CONCEPT.md): Contains the detailed visual direction, site structure, color palette, and **Image Generation Prompts**.

## Tech Stack
*   React (Vite)
*   Vanilla CSS (Variables & Glassmorphism)
*   Framer Motion (Animations)
*   Lucide React (Icons)
